# Zgine

Modern Data-Oriented 3D Game Engine

## Features

- Data-Oriented Design with ECS (EnTT)
- Modern C++17
- Cross-platform (Windows, Linux, macOS)
- Physics (Jolt Physics)
- Audio (miniaudio)
- Scripting (Lua + Sol2)
- Model Loading (Assimp)
- ImGui Integration

## Building

See [BUILD.md](BUILD.md) for detailed build instructions.

### Quick Start

#### Windows
```bash
.\build.bat debug
```

#### Linux/macOS
```bash
./build.sh debug
```
